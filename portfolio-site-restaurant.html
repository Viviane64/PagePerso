Site web pour un restaurant

Ce projet web a été réalisé dans le cadre d'une formation pour un client dans le domaine de la restauration.

L'objectif : permettre à ce restaurant de gagner en visibilité et de proposer son menu de pizza aux utilisateurs.

Le client étant novice dans les nouvelles technologies, a la possibilité de modifier facilement son menu en ligne, grâce à une interface d'administration sur-mesure qui a été réalisée dans le cadre de ce projet.

Technologies utilisées : HTML, CSS, DJANGO, PYTHON, HEROKU
